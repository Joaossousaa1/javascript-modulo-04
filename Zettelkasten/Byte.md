### **O que é um byte?**

Byte (Termo Binário) é um conjunto de 8 [[Bit]] agrupados. Cada caractere, é representado por 1 byte, ou seja, 8 bits agrupados. Byte é o espaço necessário para o armazenamento de um caractere.

| Caractere | ASCII    |
| --------- | -------- |
| A         | 00101110 |
| =         | 00111101 |
